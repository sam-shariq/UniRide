#  UniRide

UniRide is a student‑focused web application designed to make daily commuting **simpler, smarter, and more affordable**.  
It helps users plan routes, compare ride options, and split costs when traveling in groups.

---

##  What UniRide Does

- Helps students plan their commute easily
- Shows pickup and drop locations on a map
- Compares ride options based on distance and time
- Calculates estimated travel time considering traffic patterns
- Allows users to create groups and split ride costs fairly
- Provides secure login and personalized profiles

---

## Why UniRide?

- Built specifically for students
- Reduces individual travel cost through group rides
- Simple, fast, and easy to use
- Uses real logic instead of hard‑coded data
- Designed for everyday college commuting

---

##  Tech Overview

- Modern web frontend
- Lightweight backend logic
- Secure authentication
- Cloud‑based database
- Free and open‑source map services

---

##  User Flow

1. User logs in
2. Enters pickup and destination
3. Views route and estimated time
4. Compares ride options
5. Creates or joins a group
6. Cost is split automatically
